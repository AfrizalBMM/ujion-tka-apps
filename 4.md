# Module: Siswa (Test Taker)

## 1. Entry Point

- **Token Check**: Input 6 digit kode unik.
- **Profile**: Nama Lengkap (Wajib), No WA (Optional).
- **Instruction**: Page petunjuk pengisian yang interaktif.

## 2. Exam Interface (Proteksi Maksimal)

- **Forced Fullscreen**: Otomatis masuk mode fullscreen saat klik "Mulai".
- **Anti-Cheat Logic**:
    - **Desktop**: Deteksi `visibilitychange` & `blur`. Jika pindah tab, muncul modal peringatan keras.
    - **Mobile**: Deteksi `pagehide`. Jika siswa mencoba buka chat WA atau aplikasi lain, sistem mencatat log "Curang" ke Guru.
- **Navigation**:
    - Kiri: Soal + Gambar + Opsi.
    - Kanan: Nomor soal (Grid), Timer, Fitur Pembesar Teks.
    - Button: Sebelumnya, Ragu-ragu (Kuning), Selanjutnya.

## 3. Post-Exam

- **Finish Page**: Menampilkan Skor Akhir saja (Tanpa jawaban benar/pembahasan untuk keamanan soal).
